# 视频逐帧提取工具 (基于 FFmpeg)

一款基于 Python + FFmpeg 的图形化工具，可快速从视频中按时间间隔提取帧，支持自定义输出分辨率、图片格式和画质。界面采用 `customtkinter`，操作直观，适合批量处理视频素材。

---

## ✨ 功能特点

- **多种提取模式**  
  - 按时间间隔（如每 1 秒、每 0.5 秒）提取帧。  
  - 提取全部帧（极占存储，慎用）。
- **灵活的输出设置**  
  - 支持 JPG / PNG 格式。  
  - 可调画质：JPG 控制压缩率，PNG 控制压缩耗时与文件大小。  
  - 预设分辨率：保持原始、720p、1080p、2K、4K，或自定义任意宽高（建议偶数）。
- **实时日志**  
  显示 FFmpeg 执行状态，便于排查问题。
- **多线程处理**  
  界面不会卡顿，后台执行 FFmpeg 命令。
- **自动生成输出目录**  
  选择视频文件后，自动在同目录下创建 `{视频名}_frames` 文件夹。

## 🖥️ 系统要求

- Python 3.8 或更高版本
- FFmpeg（必须，且已添加到系统环境变量）
- 支持的操作系统：Windows / macOS / Linux

---

## 📦 依赖安装

需要安装以下 Python 库：

- `customtkinter` – 现代化 tkinter 扩展
- `tkinter`（Python 自带，通常无需额外安装）

安装命令：

```bash
pip install customtkinter
```

> 若没有 tkinter（多见于 Linux 最小安装），需系统级安装：  
> - Ubuntu/Debian: `sudo apt install python3-tk`  
> - macOS: 一般自带

---

## 🚀 运行程序

将代码保存为 `extractor.py`，然后执行：

```bash
python extractor.py
```

## 🧭 使用指南

### 1. 选择视频文件

点击 **“浏览文件”** 或手动输入视频路径，支持常见格式：`.mp4`, `.avi`, `.mkv`, `.mov`, `.flv`, `.wmv` 等。

选择后，输出目录会自动填入 `{视频文件名}_frames`（可手动更改）。

### 2. 设置输出目录

点击 **“选择目录”** 指定提取的图片保存位置。

### 3. 提取设置

- **按时间间隔**：输入秒数（支持小数，如 `0.5`），每隔该秒数提取一帧。
- **提取所有帧**：将视频的每一帧都输出（文件数量巨大，请确保磁盘空间充足）。

### 4. 图片格式

- **JPG**：有损压缩，文件较小，适合预览。
- **PNG**：无损压缩，文件较大，适合需要透明或高保真的场景。

### 5. 图片画质

- **JPG 模式**：  
  - 最高质量 → 质量参数 `1`（文件最大）  
  - 低占用 → 质量参数 `8`（文件最小，画质下降明显）
- **PNG 模式**：  
  - 最高质量 → 压缩级别 `0`（最快，文件大）  
  - 低占用 → 压缩级别 `9`（最慢，文件小）  
  对 PNG 画质无损，仅影响压缩时间与最终文件体积。

### 6. 输出分辨率

- **保持原始**：不缩放。
- **预设**：720p/1080p/2K/4K 自动缩放（宽度按比例，高度自动计算）。
- **自定义**：需手动输入宽度和高度（建议偶数，如 `1920×1080`）。

### 7. 开始提取

点击 **“▶ 开始批量提取”**，程序会在后台运行 FFmpeg，日志区域显示进度（不会逐行打印过多信息，以免卡顿）。

提取完成后，日志区会显示成功或错误信息。

## ⚠️ 注意事项

- **必须安装 FFmpeg**：  
  - Windows：下载 [FFmpeg](https://ffmpeg.org/download.html) 并添加到 `PATH`。  
  - macOS：`brew install ffmpeg`  
  - Linux：`sudo apt install ffmpeg`  
  若未安装，程序会提示 “找不到 FFmpeg”。
- **输出目录**：如果目录不存在会自动创建，但请确保有写入权限。
- **帧数预估**：提取所有帧可能导致生成成千上万张图片，请注意磁盘空间。
- **宽高偶数**：FFmpeg 的缩放过滤器推荐使用偶数宽高，否则可能报错。程序已做校验。
